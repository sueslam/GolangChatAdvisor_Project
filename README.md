# Golang_Chat_Advisor
Build a backend service to create an AI advisor where client can perform the following tasks. 
The goal of this project is to showcase backend design, session handling, API design and AWS deployment.
- create a companion
- start a chat session
- send a message
- get a mocked AI reply
- fetch session history

Tech Stack Used: 
- Go
- AWS Lambda
- API Gateway
- DynamoDB
- AWS SAM
- GitHub Actions
